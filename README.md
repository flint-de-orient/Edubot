# EduBot - Educational Learning Chatbot

## Setup

### Backend (Flask)
1. Install Python dependencies:
```bash
pip install -r requirements.txt
```

2. Run the Flask backend:
```bash
python app.py
```

### Frontend (React + Vite + Tailwind CSS)
1. Install Node.js dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open http://localhost:3000 in your browser

## Features
- 📚 AI-powered educational assistance using Gemini API
- 🎓 Interactive learning across multiple academic subjects
- 📊 Learning progress tracking
- 🌱 Subject suggestions and educational facts
- 💬 Modern React UI with Tailwind CSS
- 🚀 Fully AI-driven responses with no fallback modes

## Subjects Covered
- Mathematics & Problem Solving
- Science & Scientific Concepts
- History & Historical Events
- Literature & Language Arts
- Geography & World Knowledge
- Technology & Computer Science
